# Android-ELRS Converter

**Lightweight RC Solution for Beginners**

![Android-ELRS Product](/manuals/Android-ELRS-manual/assets/1.png)

## Overview

The Android-ELRS Converter is an innovative hardware that transforms your Android phone into an RC transmitter. Compared to traditional dedicated RC transmitters, using a phone as a controller can lower the entry cost, making it suitable for beginners and lightweight flying.

## Key Features

- **Plug and Play** - Powers directly from phone's Type-C port, no external battery needed
- **Open Source Link** - Based on ExpressLRS (ELRS) open source project with good refresh rate and anti-interference capability
- **Dual Control Modes** - Supports both virtual joystick and motion control
- **Integrated Charging** - Built-in 1S battery charging system with 500mA constant current charging

## Technical Features

- Uses serial conversion chip to ensure communication stability
- Integrates ExpressLRS wireless link for reliable RC performance
- Built-in antenna meets daily flying distance needs
- Supports Crossfire (CRSF) protocol standard

## Use Cases

- RC beginners learning and practice
- Lightweight fixed-wing aircraft control
- Small multirotor drone operation
- Indoor/outdoor recreational flying
